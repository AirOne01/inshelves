# Inshelves

I'm pretty disapointed at Steam all the videogame clients and stores, so I decided to make mine.

## (Not) Current state

![Demo](https://i.imgur.com/zHpZXHU.png)

## Try it yourself

```bash
# clone the repo
$ git clone https://github.com/AirOne01/inshelves
$ cd inshelves

# install the dependencies
$ yarn

# run the dev server
$ yarn dev
```
